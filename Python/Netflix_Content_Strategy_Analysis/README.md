# Netflix Content Strategy Analysis

## Project Overview

This project analyzes Netflix's content catalog using Python and Exploratory Data Analysis (EDA).

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Questions

- Does Netflix focus more on Movies or TV Shows?
- Which countries contribute the most content?
- Which genres dominate Netflix?
- How has Netflix grown over time?

## Key Findings

- Movies account for the majority of Netflix content.
- Significant growth occurred after 2015.
- United States is the largest content contributor.
- Drama and International Movies dominate the platform.
