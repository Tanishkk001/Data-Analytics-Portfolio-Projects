# 🛒 Superstore Sales Analysis using SQL

## 📌 Project Overview
This project analyzes the Sample Superstore dataset using SQL to uncover business insights related to sales, profit, customers, products, and regional performance. The project demonstrates data querying, aggregation, filtering, and analytical problem-solving skills using MySQL.

---

## 🎯 Objectives
- Analyze overall sales and profit performance.
- Identify top-performing customers and products.
- Study regional and category-wise performance.
- Discover trends and patterns in the business data.
- Generate actionable insights through SQL queries.

---

## 🛠️ Tools & Technologies
- MySQL
- MySQL Workbench
- SQL
- Git & GitHub

---

## 📂 Dataset
**Dataset:** Sample Superstore Dataset

The dataset contains information related to:
- Orders
- Customers
- Products
- Sales
- Profit
- Categories and Sub-Categories
- Regions and States
- Order Dates

---


## 🔍 Business Questions Solved

1. What are the total sales and total profit?
2. Which regions generate the highest revenue?
3. Which product categories perform the best?
4. Which customers contribute the most sales?
5. Which products generate the highest profit?
6. Which states have the highest and lowest sales?
7. What are the monthly sales trends?
8. Which sub-categories are underperforming?

---

---

## 💡 Key Insights

- Technology products generated the highest revenue.
- A small percentage of customers contributed significantly to total sales.
- Some sub-categories produced high sales but low profitability.
- Sales performance varied considerably across different regions and states.
- Monthly trend analysis helped identify seasonal demand patterns.
